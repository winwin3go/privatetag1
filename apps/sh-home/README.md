# sh-home

StrongHold cloud-side home memory portal skeleton.

## Contract

- **Routes**: `/home`, `/vault/*`, `/api/memories/*`.
- **Responsibilities**:
  - Render StrongHold memory timelines and vault details using `pkg/sh-domain`.
  - Fetch Tag/Action metadata via `svc/tag-core` when a memory references PrivateTag assets.
  - List and view media via `svc/media-core` using memory-attached `mediaId`s.
- **Auth**: Enforces StrongHold household + member identities validated by `svc/st-idp`.
- **Integrations**: Publishes audit events for sensitive reads to `svc/audit-core`; may subscribe to notifications generated by `svc/np-core`.

Skeleton only; implementation will be added later.
